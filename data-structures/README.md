# Data Structures

This track is for C++ implementations and experiments with core data
structures.

Suggested folder shape:

```text
data-structures/
  hash-table/
    README.md
    include/
    tests/
  binary-tree/
    README.md
    include/
    tests/
```

Register test targets in this directory's `CMakeLists.txt`, then run the root
test helper:

```powershell
.\scripts\test.ps1
```
