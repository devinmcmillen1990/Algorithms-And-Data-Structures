# GeeksforGeeks

This track is for C++ practice problems from GeeksforGeeks.

Suggested folder shape:

```text
geeksforgeeks/
  problem-slug/
    README.md
    PROBLEM.md
    SOLUTIONS.md
    _md_images/
    solutions/
    tests/
```

Register test targets in this directory's `CMakeLists.txt`, then run the root
test helper:

```powershell
.\scripts\test.ps1
```
